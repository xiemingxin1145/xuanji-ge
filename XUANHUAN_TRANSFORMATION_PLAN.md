## 一、核心愿景（零已确认）

把现有的「玄机阁」玄学工具App，全面升级为**玄幻 + 神话 + 天时地利战略 + 多功能游戏元素**的沉浸式神话APP。

**新世界观**：
上古神话时代，玄机阁是连接天庭、人间、幽冥、妖域、仙界的永恒枢纽。玩家觉醒成为新任「玄机阁主传人」，通过推演「玄机神通」影响三界命运、突破自身道行、布局势力。

所有原有工具都升级为上古神通的具现：
- 罗盘 = 仙域导航与战场布阵神通
- 八字 = 命运天书与羁绊神通
- 易经起卦 = 神谕决策与战局推演
- 黄历 = 三界时令与借势神通
- 姓名五行 = 灵名炼字与召唤/炼器
- AI玄师 = 上古玄天残识（有记忆的古神智脑）

**核心创新机制**：**天时·地利·人和「玄机共鸣」系统**

开发共享引擎：
- 天时（黄历吉凶）
- 地利（罗盘方位/地形匹配）
- 人和（八字羁绊 + 姓名五行相生）
三者叠加触发「玄机共鸣」爆发（视觉特效 + 数值暴击 + 剧情加成）。

用于：修炼突破、战斗结算、重大决策、探索隐藏秘境、势力布局。
